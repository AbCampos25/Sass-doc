O que é Sass?

Sass, Syntactically Awesome Style S heet, ou folha de estilo sintaticamente incrível em uma tradução direita, é uma maneira de escrever as regras css realmente incrível.

Torna possível a criação de variáveis e até funções que podem ser reutliz adas como se fosse em uma linguagem de progamação mesmo. Até o conceito de oo se aplica em sass.

Extensão do arquivo

A extensão do sass é .scss "nomeArquivo.scss".

Para que o navegador leia o que que há no .scss (Super Css) é necesário que se trasforme o arquivo .scss em .css, e isso não é problema nenhum porque a transformação é feita de forma automática, para isto basta que instale uma extensão no editor de código compatível com sass, ou seja, que possa compilar e fazer a transformação.

Para o VS code, a extensão seria o "live compiler Sass" que é simples de ser instalado.

Nota: Ao importar o arquivo no doc html, a extensão do arquivo mantem-se como css "nommeArquivo.css", isso porque o navegador não lê arquivos.scss.

Declaração de variáveis e sua utilização

A declaracão de variáveis é feita através do "$" como em php.

$color1:rgb(250,250,250);
$bgColor:#202020;
*{
box-sizing: border-box;
color: $color1;
margin: 0; padding: 0; }
body{
padding: 30px;
background: $bgColor; }

Selecionar elementos dentro de outros elementos e declarar suas regras css

Com o Sass é possível declarar as regras de um elemento filho dentro das regras do elemento pai.

div.content{
margin: 10px 0;
text-align: justify;
h1{
color: $bgColor;
background-color: $color1; } }

Pseudo-propriedades

A declaração de pseudo-propriedades é feita dentro das regras do elemento "&:hover{...}" como no se vê no exemplo.

button{
border-radius: 5px;
font-size: 14pt;
padding: 5px;
display: block;
color: red;
margin-top: 20px;
&:hover {
border: 1px solid;
background-color: transparent;
&::after{
content: "1"; } } }

Importar arquivos

1. O arquivo a ser imporatado deve _nomeArquivo.scss.

2. Para importar, basta que se declare o diretório do arquivo + o arquivo, da seguinte forma:
@import './header';

$color1:rgb(250,250,250);
$bgColor:#202020;
@import 'header';
@import 'content';
*{
box-sizing: border-box;
color: $color1;
margin: 0;
padding: 0;
}
body{
padding: 0 30px;
background: $bgColor;
}

Mixim or Function

É possível criar funções, com ou sem parámetro, e chamá-las onde for necessário como se vê nos 2 primeiros exemplos.

Também é possível extender as regras de um arquivo noutro usando o @extend como no último exemplo.

@mixin flexbox(){
display:flex;
flex-flow: row;
justify-content: center;
}
header{
@include flexbox();
padding: 10px;

font-weight: bold;
